Windows notes:

Prequisites:
 - Visual Studio (or some other source for cl.exe)
 - MSYS2
  - diffutils (pacman -S diffutils)
  - make (pacman -S make)
 - YASM

FFmpeg compile Instructions:
 - Clone ffmpeg, checkout n4.2.2
 - Place yasm.exe (rename it) into the ffmpeg folder
 - Run VS command prompt
 - From the VS command promp run msys2_shell.cmd -use-full-path
 - Navigate to ffmpeg checkout
 - export PATH="$PWD:$PATH" (this is for yasm, could just put it somewhere else on the path)
 - Run configure (add --toolchain=msvc)
 - make, make install
 - Put build_output into the builds folder renaming it for the target triple (Ex: x86_64-pc-windows-msvc)
